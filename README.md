
# 💸 SpendSensei 
Personal finance manager for students 

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Database:** SQLite3
* **Frontend:** HTML5, Vanilla JavaScript, Custom CSS
* **Libraries:** Chart.js

## 🚀 How to Run Locally
1. Clone the repository: `git clone https://github.com/YOUR_USERNAME/SpendSensei.git`
2. Navigate to the folder: `cd SpendSensei`
3. Install requirements: `pip install Flask`
4. Run the app: `python app.py`
5. Open your browser and go to `http://127.0.0.1:5000`
